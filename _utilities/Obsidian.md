Organising files:
____tags: Folder for all the tags
_templates: Folder for USEFUL templates for day to day tasks
_utilities: Folder for shot-cuts and quick copy and past

The folder structure should be as follows
	**Year-folder** -> **Year-files**, All_md_files

The properties of each file should also be kept as:
	tags: they define the theme and/or topic of each note
	type: pre-rough, rough, planned

File naming should'nt be more than 4word
